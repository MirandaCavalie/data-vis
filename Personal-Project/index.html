Displaced — 27 Years of Eviction in San Francisco

This project explores 48,000+ eviction notices filed in San Francisco from 1997 to 2026. Using scrollytelling and interactive dashboards built with D3.js, it reveals how displacement patterns shifted across neighborhoods and economic cycles.

Project Title: Displaced: 27 Years of Eviction in San Francisco
Name: Miranda Cavalie
Dataset: SF Rent Board Eviction Notices
Dataset Portal: DataSF

Planned Visualizations

Visualization 1

Choropleth Map — Eviction Density by Neighborhood

Objectives Obj 1 — Temporal Trends Obj 2 — Geographic Displacement
Data Neighborhoods - Analysis Boundaries, File Date (year), eviction count per neighborhood per year
D3 d3.geoMercator(), d3.geoPath(), d3.scaleSequential(d3.interpolateBlues), GeoJSON bindings

Visualization 2

Line Chart — Evictions Over Time (1997–2026)

Objectives Obj 1 — Temporal Trends Obj 4 — Policy Impact

D3 d3.line(), d3.scaleTime(), d3.scaleLinear(), d3.brushX() for year-range selection

Visualization 3
